Installing Adspaces Plugin

There are two main methods of installing the Adspaces plugin

First Method

  1. Download the plugin

    If you already have the plugin zipped file, you can skip this step.

  2. Navigate to the plugins menu in your WordPress installation

  3. Click "Add New"

  4. Click "Upload Plugin"

  5. Then choose the zip file downloaded before and click "Install Now"

  6. Activate the plugin from the Plugins menu within the WordPress admin

Second Method

Second method of manual installation involves downloading our Adspaces plugin and uploading it to your web-server via your favorite FTP application.

  1. Download the plugin

    If you already have the plugin zipped file, you can skip this step.

  2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installationâ€™s wp-content/plugins/ directory.

  3. Activate the plugin from the Plugins menu within the WordPress admin

Once installation is done, go to the adspaces plugin settings page

Fill in the fields

  1. Organisation

    Enter the Name of your Organisation

  2. Background color

    Enter the background color of the web page in which you want to embed the Adspaces plugin. Colors should be entered in the form 000000 for black FFFFFF for white, FF0000 for red and so on, i.e do not include a leading "#" character.

  3. Header Color

    Input the color to use for headers and buttons in the Adspaces plugin. Use the format described in "2" above (no leading "#" character).

  4. Text Color

    Enter the text color for the Adspaces plugin. Color values should folloe the format described above.

  5. Font Family

    Enter the font type to be used by the plugin, for example Droid Serif.

  6. Enter the font weight
  7. Enter the font size, including the units, e.g 14px, 14em, 14px, 14em...

Try to chose values that will make the Adspaces plugin blend in with the rest of your website. When you're done, click "Save" and then fill out the Authentication form below.

Authentication

Enter your AdSpaces Application Client ID and your AdSpaces Application Client Secret.


Embed the Adspaces plugin in your page

When installation is complete, you'll need to embed the Adspaces plugin in your webpage:

  1. Go to "Pages" on the WordPress dashboard

    Select the page in which you want to embed the plugin and click "Edit"
